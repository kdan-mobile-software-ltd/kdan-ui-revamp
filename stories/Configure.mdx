## Getting Started

#### 1. Installation
```sh
 yarn add kdan-ui-revamp@latest
```

#### 2. Wrap `<KdanUiThemeProvider>` for Kdan UI theming
```jsx
// _app.tsx
import { KdanUiThemeProvider } from 'kdan-ui-revamp';

const App = ({ Component, pageProps }: AppProps) => {
   return (
      <KdanUiThemeProvider>
        <Component {...pageProps} />
      </KdanUiThemeProvider>
    );
};

export default App;
```

#### 3. [optional] 如果「目的專案」未設定 css reset，可以使用 kdan-ui 的 `kdanUiResetCss`。

以 styled-components 的 `createGlobalStyle` 初始化為範例：

in 「目的專案」 `pages/_globalStyle.tsx`

```jsx
import { createGlobalStyle } from 'styled-components';
import { kdanUiResetCss } from 'kdan-ui-revamp';

const GlobalStyle = createGlobalStyle`
  ${kdanUiResetCss} /* <- reset css */
`;

export default GlobalStyle;
```

in 「目的專案」 `_app.tsx`
```jsx
...
import GlobalStyle from './_globalStyle';

const App = ({ Component, ...rest }: AppProps) => {
  const { store, props } = wrapper.useWrappedStore(rest);
  return (
    <Provider store={store}>
      <KdanUiThemeProvider>
        <GlobalStyle /> {/* <- use styled-components syntax */}
        <Component {...props.pageProps} />
      </KdanUiThemeProvider>
    </Provider>
  );
};
```

## Quick Usage

```jsx
import React from 'react';
import { Button } from 'kdan-ui-revamp';

function Page() {
  return (
    <Button
      corner="round"
      size="large"
      variant="filled"
      onClick={() => { console.log('hello kdan!')}}
    >
      buttonText
    </Button>
  );
}
```

## For Users

### 重要版更變動（depreciation notification）

#### `2.0.0-beta.3` 之後

* `<Banner>`, `<CtaBanner>` 的 button 傳入方式，從 data 改為 children。


#### `2.0.0-beta.2` 之後

* `<Button>` 元件不支援 `color`, `backgroundColor`, `borderColor`, `hoverColor`, `hoverBackgroundColor`, `hoverBorderColor` 等等 props，一律改成 `CustomCss` props 開放自行寫 css 的彈性。




## For Infra Developers

* 正式上線 (npm publish) 前要更新 package.json 的 `version` 版號
* 要 export 的 components 要在 `index.tsx` 寫上
* export 的 component file 要使用 named export 而非 default export
